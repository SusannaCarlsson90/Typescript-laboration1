Laboration 1: Kursadministration

https://typescript-laboration1.onrender.com/

Skapat en webbapplikation för att hantera kurser med kurskod, kursnamn, progression samt länk till kursplan. Byggd med TypeScript och Vite, där jag även använt mig av localStorage för att lagra kursdata mellan sessioner, och DOM-manipulation för att visa kursinformation på webbsidan.
